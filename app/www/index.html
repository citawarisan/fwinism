Instructions

  1. Prepare a ESP32 flashed with micropython
  2. Copy the scripts in esp32 directory to ESP32
  3. Run the ESP32, connect to the microcontroller's network (ssid: citawarisan, password: citawarisan)
  4. Click the button below to start experimenting with the app!

FAQ

Q: Why is the app not working?

A: Make sure you have connected to the ESP32's network

Q: What if I don't have an ESP32 / too lazy to setup one?

A: There's, in fact a dummy preview for the app, but it is non-functional
